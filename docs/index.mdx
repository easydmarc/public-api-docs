    <div className="section-description">Get up and running with the EasyDMARC API in minutes.</div>

    <div className="quickstart-step">
      <div className="step-number">1</div>
      <h3>Get your API key</h3>
      <p>Log in to your EasyDMARC partner dashboard and generate an API key from the settings page.</p>
    </div>
    <div className="quickstart-step">
      <div className="step-number">2</div>
      <h3>Authenticate requests</h3>
      <p>Include your API key in every request using the <code>Authorization: Bearer &lt;token&gt;</code> header.</p>
    </div>
    <div className="quickstart-step">
      <div className="step-number">3</div>
      <h3>Make your first call</h3>
      <p>Try a DNS lookup or list your organizations to verify everything is working correctly.</p>
    </div>
  </div>
  <div className="quickstart-code">

```bash title="Example: DNS lookup for a DMARC record"
curl -X GET "https://api.easydmarc.com/v1.0/dns/dmarc?domain=example.com" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

  </div>

  <div className="quickstart-response">

```json title="Response"
{
  "data": {
    "dmarc": "v=DMARC1; p=reject; rua=mailto:rua@example.com; ruf=mailto:ruf@example.com; pct=100"
  },
  "meta": {
    "domain": "example.com",
    "record_type": "dmarc"
  }
}
```

  </div>
